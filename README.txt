---------------------------Sonic Generations Mod--------------------------
-----------------------------Sonic 06 HUD v2.2----------------------------

This mod directly imports or recreates HUD from Sonic The Hedgehog (2006) by using XNCP files directly from the original game or using ImGui. This HUD mod is fully compatible with Sonic Generations, including Classic/Modern Sonic, Missions and Boss battles.

This mod is also compatible with "Score Generations" mod, you can download it here: https://gamebanana.com/mods/292798
To use the score system properly on Generations or Unleashed Project stages, released together with this mod, please enable "Sonic 06 Score System (Generations Stages)" or "Sonic 06 Score System (Unleashed Project)" mod.

Mod priority should be the following:
-Sonic 06 HUD
-Sonic 06 Definitive Experience
-Sonic 06 Score System
-Score Generations
-Level Mods

Only works with HedgeModManager, SonicGMI is NOT supported!!!

------------------------------------------------------------
Mod Configurations
------------------------------------------------------------
Character Life Icon:
-Set which character life icon to use (10 in total), this option is ignored if "Sonic 06 Definite Experience" mod is used

Enable Character UI Color:
-Use unused character UI color, only works for Sonic, Shadow and Silver, other characters will use Sonic's color

------------------------------------------------------------
Features
------------------------------------------------------------
-Direct gameplay HUD import from Sonic 06
-Boss health bar in boss battles
-Recreated 06 result screen
-06 Text Box and font (used for Omochao & cutscenes)
-06 Score System (requires "Score Generations" mod enabled)
